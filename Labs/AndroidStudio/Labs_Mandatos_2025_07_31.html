Ejercicio: Implementación de un Splash Screen Animado

Aprende a añadir una pantalla de bienvenida profesional a tu aplicación usando la API oficial de Splash Screen, un ícono vectorial y una animación simple.

  • SplashScreen API: La librería oficial de Android para crear pantallas de bienvenida compatibles con todas las versiones del sistema.
  • VectorDrawable y AnimatedVectorDrawable: Formatos XML para crear íconos escalables y animaciones eficientes.
  • Temas (Themes): Cómo usar un tema específico para controlar la apariencia del splash screen antes de que la app se cargue.
  • ViewModel y StateFlow: Para gestionar el estado de "carga" y decidir cuándo debe desaparecer el splash screen.

Prepara tu proyecto añadiendo las dependencias necesarias a través del Version Catalog.

  • 1.1. Abre libs.versions.toml:
    • En la sección [versions], añade las claves para coreSplashscreen y lifecycleViewmodelCompose.
    • En la sección [libraries], define los alias para androidx-core-splashscreen y androidx-lifecycle-viewmodel-compose, referenciando las versiones anteriores.
  • 1.2. Abre build.gradle.kts (módulo app):
    • En el bloque dependencies, añade las librerías usando los alias del catálogo (ej. implementation(libs.androidx.core.splashscreen)).
  • 1.3. Sincroniza tu proyecto con Gradle.

Crea los recursos visuales para tu splash screen.

  • 2.1. Crea el Ícono Vectorial:
    • Usa el "Vector Asset Studio" de Android Studio (File > New > Vector Asset) para elegir un ícono del catálogo. Nómbralo ic_splash_logo.xml.
    • Ajusta el tamaño del lienzo: Edita el archivo XML del ícono. Cambia android:width y android:height a 288dp.
    • Escala y Centra: Envuelve los <path> de tu ícono dentro de un <group>. Usa los atributos scaleX, scaleY, y pivotX/Y en este grupo para escalar tu ícono hasta que se vea bien centrado y de un tamaño adecuado en la ventana de "Preview".
  • 2.2. Crea la Animación:
    • Crea un directorio de recursos res/animator.
    • Dentro, crea un archivo XML (p. ej., anim_splash_fade_in.xml). Define una animación simple usando objectAnimator. Una buena opción es animar la propiedad alpha (transparencia) de 0 a 1 para un efecto de aparición suave.
  • 2.3. Conecta el Ícono y la Animación:
    • Crea un archivo AnimatedVectorDrawable en res/drawable (p. ej., avd_splash_logo.xml).
    • Dentro de este archivo, usa la etiqueta <animated-vector> para enlazar tu @drawable/ic_splash_logo con un <target> que apunte al grupo de tu ícono y a tu @animator/anim_splash_fade_in.

Define la apariencia del splash screen a través de un tema personalizado.

  • 3.1. Crea un Tema para el Splash:
    • En tu archivo res/values/themes.xml, crea un nuevo <style>.
    • Dale un nombre (ej. Theme.App.Starting) y haz que herede de Theme.SplashScreen.
  • 3.2. Configura los Atributos: Dentro del tema, establece los siguientes <item>:
    • windowSplashScreenBackground: El color de fondo.
    • windowSplashScreenAnimatedIcon: Tu recurso @drawable/avd_splash_logo.
    • windowSplashScreenAnimationDuration: La duración en milisegundos de tu animación.
    • postSplashScreenTheme: El tema principal de tu aplicación que se usará después del splash.
  • 3.3. Aplica el Tema en el Manifiesto:
    • En tu AndroidManifest.xml, asigna el nuevo tema del splash screen al atributo android:theme de tu etiqueta <application> y de tu MainActivity.

Gestiona el estado de carga de tu aplicación para decidir cuándo ocultar el splash screen.

  • 4.1. Crea un MainViewModel:
    • Crea una nueva clase Kotlin que herede de ViewModel.
  • 4.2. Define un Estado de Carga:
    • Dentro del ViewModel, crea un MutableStateFlow<Boolean> llamado _isLoading con un valor inicial de true.
    • Expón una versión inmutable (StateFlow) llamada isLoading.
  • 4.3. Simula la Carga:
    • En el bloque init del ViewModel, usa viewModelScope.launch para iniciar una corrutina.
    • Dentro de la corrutina, espera unos segundos usando delay() y luego cambia el valor de _isLoading a false.

Conecta todos los componentes en tu actividad principal.

  • 5.1. Inicializa el ViewModel: En tu MainActivity, obtén una instancia de tu MainViewModel usando la delegación by viewModels().
  • 5.2. Instala el Splash Screen: En el método onCreate, **antes** de llamar a super.onCreate() o setContent, llama a installSplashScreen().
  • 5.3. Mantén el Splash en Pantalla:
    • Usa el método splashScreen.setKeepOnScreenCondition.
    • La lambda de esta condición debe devolver el valor actual de tu estado viewModel.isLoading.value.
  • 5.4. (Opcional) Personaliza la Animación de Salida:
    • Usa splashScreen.setOnExitAnimationListener para definir una animación personalizada.
    • Dentro del listener, puedes animar propiedades de splashScreenView.iconView como SCALE_X, SCALE_Y, y ALPHA usando ObjectAnimator.
    • Asegúrate de llamar a splashScreenView.remove() cuando tu animación termine (puedes usar doOnEnd).
